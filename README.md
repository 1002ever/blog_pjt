## :pencil2: 서비스 소개

*  **'#' 기능으로 항목별 자소서 관리**
   * Ex) 지원동기, 장단점
*  **나만의 채용 공고**
   * 타임 테이블
   * 채용 진행 상황

*  **카테고리별 최신 뉴스**

*  **간단한 한 줄 메모**
